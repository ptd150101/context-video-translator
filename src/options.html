Realtime Video Subtitle Translator

OpenAI-compatible subtitle translation for YouTube and Udemy, with ahead-of-time batching, karaoke timing, furigana, and IndexedDB cache.

YouTube + Udemy

Provider

Realtime behavior

On YouTube, press Alt+E to edit mask/subtitle regions directly on the video. Press Alt+H to toggle the hard-sub mask.

Udemy support

Udemy adapter tries network VTT/API discovery first, then native video textTracks/track elements. If a lecture has no captions, the overlay will show a no-captions message.

Subtitle layout

The manual region now controls the anchor position and width. Height is measured from the rendered subtitle, so furigana and 2-line subtitles should not be truncated.

Japanese study mode

Study mode runs locally. Furigana uses Kuromoji when available and falls back to the lightweight rule engine while the dictionary loads. Karaoke progress tracks the whole original subtitle line, so it stays clean instead of lighting many tiny grammar chunks.

How to test

  1. Open a YouTube video or a Udemy lecture that has captions.
  2. On YouTube, turn captions on once so timedtext is requested. On Udemy, open the lecture and enable captions if they do not load automatically.
  3. The extension parses the caption track, translates upcoming cues in batches, and renders its own overlay.

For local servers, use LM Studio, llama.cpp, Ollama-compatible OpenAI proxy, vLLM, or any endpoint that supports /v1/chat/completions.

Layout shortcuts: Alt+E edit layout, Alt+H toggle hard-sub mask, Alt+S save current video layout, Tab switch selected region, arrow keys move, Shift+Arrow resize.
